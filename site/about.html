About

This is a sample website made with soupault.

You can use it as a template for your own website, or simply learn from its setup.

  • The configuration is in the soupault.conf file.
  • Page source files are in site/.
  • Page template is in templates/main.html.
  • Navigation menu link highlighting is done by the section-link-highlight plugin.
  • Lua plugins are stored in plugins/. You can find more plugins at soupault.neocities.org/plugins.

Note that this is intentionally a "no moving parts" setup.

One of the special powers of soupault is that it can use external programs as data sources. For example, you can use a Markdown processor of your choice to convert *.md pages to HTML before further processing. You can also run the content of <code> elements through any syntax highlighter, insert output of programs into your pages and more.

Of course you will need those programs in your build environment. This setup is intentionally limited to built-in features and Lua plugins so that it will work on any environment. Consult the documentation to learn how to integrate soupault's workflow with external programs.
